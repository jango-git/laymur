Aspect & Proportion Constraints

This example demonstrates maintaining aspect ratios and creating proportional layouts using constraint-based positioning.

Constraint Types

// Aspect ratio constraint new UIAspectConstraint( element, { ratio: 16/9 } ); // Horizontal proportion constraint new UIHorizontalProportionConstraint( parent, element, { proportion: 0.5 } ); // Vertical proportion constraint new UIVerticalProportionConstraint( parent, element, { proportion: 0.3 } );

How It Works

  • Aspect Constraint: Maintains width/height ratio regardless of size changes
  • Proportion Constraints: Position elements as percentage of parent size
  • Responsive: Automatically adapts to container resizing
